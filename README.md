run command:
1. cmake -G "Visual Studio 17 2022" -B build
2. cd build
3. run FSM.sln
-----------------------------------------
without vs solution:
1. mkdir build, cd build
2. cmake ..
3. cmake --build .
4. open unitTests, run the unitTests.exe to test execution
